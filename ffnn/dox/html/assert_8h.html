Compile-time assert to enforce templated layer instancing pattern.

All layer object (except for the Layer base-class) take the following template arguments: 

*        template <typename ValueType,
*                  ...
*                  typename Options,
*                  typename Extrinsics>
*

where the Extrinsics parameter is reserved for a compile-time structure used to deduce types used by the parent layer class. Types are deduced using the ValueType and Options structure, which represent the coefficient type and compile-time sizing options, respectively.
