Compile-\/time assert to enforce templated layer instancing pattern. 

All layer object (except for the Layer base-\/class) take the following template arguments\-: \begin{DoxyVerb}*        template <typename ValueType,
*                  ...
*                  typename Options,
*                  typename Extrinsics>
*        \end{DoxyVerb}
 where the {\ttfamily Extrinsics} parameter is reserved for a compile-\/time structure used to deduce types used by the parent layer class. Types are deduced using the {\ttfamily Value\-Type} and {\ttfamily Options} structure, which represent the coefficient type and compile-\/time sizing options, respectively. \hypertarget{assert_8h_a2b49bd7cbe87713cc51996494f3c4fd5}{\index{assert.\-h@{assert.\-h}!F\-F\-N\-N\-\_\-\-S\-T\-A\-T\-I\-C\-\_\-\-A\-S\-S\-E\-R\-T\-\_\-\-M\-S\-G@{F\-F\-N\-N\-\_\-\-S\-T\-A\-T\-I\-C\-\_\-\-A\-S\-S\-E\-R\-T\-\_\-\-M\-S\-G}}
